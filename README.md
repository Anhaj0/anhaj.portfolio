- Hosted on GitHub Pages

## Installation & Deployment

No installation required. The website is static and can be hosted on any static web hosting platform.

To deploy your own copy:

1. Clone this repository:
   ```bash
   git clone https://github.com/Anhaj0/anhaj.portfolio.git
````

2. Customize the content as needed.
3. Push your changes to your GitHub repository.
4. Enable GitHub Pages in repository settings to host the site.

## Usage

Open `index.html` in any modern web browser to view the portfolio locally.
